# EcommerceProject
Full-Stack E-Commerce Web Application built using React, Tailwind CSS, Node.js, Express, and MongoDB. Includes user authentication, admin dashboard, product management, categories, cart, wishlist, orders, payments, and secure password reset with email OTP. Fully ready for deployment on Render (backend) and Vercel (frontend).
